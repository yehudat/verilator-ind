Potential issues (with solutions):

1. Build timeout
   ‚Üí Use Dockerfile.quick

   ‚Üí Edit Dockerfile: change "make -j$(nproc)" to "make -j2"

   ‚Üí X11 forwarding not set up

   ‚Üí Check DISPLAY variable

   ‚Üí Run: chmod -R 755 project/

   ‚Üí Check internet connection
   ‚Üí GitHub must be accessible for full build
